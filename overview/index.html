Quick Start

  • Get msm.hpp header
    wget https://raw.githubusercontent.com/krzysztof-jusiak/msm-lite/master/include/msm/msm.hpp
  • Include the header
    #include "boost/msm/msm.hpp"
  • Compile with C++14 support
    $CXX -std=c++14 ...
